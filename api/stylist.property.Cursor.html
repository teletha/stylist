Cursor()

defaults()Cursor

Cursor

Default cursor, typically an arrow.

none()Cursor

Cursor

No cursor is rendered

contextMenu()Cursor

Cursor

A context menu is available under the cursor. In Gecko/Firefox not implemented on Windows, bug 258960 WONTFIX.

help()Cursor

Cursor

Indicating help is available.

pointer()Cursor

Cursor

E.g. used when hovering over links, typically a hand.

progress()Cursor

Cursor

The program is busy in the background but the user can still interact with the interface (unlike for wait).

waiting()Cursor

Cursor

The program is busy (sometimes an hourglass or a watch).

cell()Cursor

Cursor

Indicating that cells can be selected.

cross()Cursor

Cursor

Cross cursor, often used to indicate selection in a bitmap.

text()Cursor

Cursor

Indicating text can be selected, typically an I-beam.

verticalText()Cursor

Cursor

Indicating that vertical text can be selected, typically a sideways I-beam.

alias()Cursor

Cursor

Indicating an alias or shortcut is to be created.

copy()Cursor

Cursor

Indicating that something can be copied.

grab()Cursor

Cursor

Something can be grabbed (dragged to be moved).

grabbing()Cursor

Cursor

Something is being grabbed (dragged to be moved).

move()Cursor

Cursor

The hovered object may be moved.

noDrop()Cursor

Cursor

Cursor showing that a drop is not allowed at the current location.

notAllowed()Cursor

Cursor

Cursor showing that something cannot be done.

allScroll()Cursor

Cursor

Cursor showing that something can be scrolled in any direction (panned).

colResize()Cursor

Cursor

The item/column can be resized horizontally. Often rendered as arrows pointing left and right with a vertical bar separating.

rowResize()Cursor

Cursor

The item/row can be resized vertically. Often rendered as arrows pointing up and down with a horizontal bar separating them.

nResize()Cursor

Cursor

Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box.

eResize()Cursor

Cursor

Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box.

sResize()Cursor

Cursor

Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box.

wResize()Cursor

Cursor

Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box.

neResize()Cursor

Cursor

Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box.

nwResize()Cursor

Cursor

Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box.

seResize()Cursor

Cursor

swResize()Cursor

ewResize()Cursor

nwseResize()Cursor

Cursor
